# Food Mela - Leftover Food Marketplace

## Description
Food Mela is an initiative aimed at reducing food wastage by redistributing surplus fresh food from local restaurants at affordable prices. This platform enables users to book discounted coupons for meals, partner with restaurants, and contribute to minimizing food wastage.

## Features
- Book discounted coupons for meals from local restaurants.
- Partner with us as a restaurant and reach a wider audience.
- Easy checkout process for ordering meals.
- Check out with the "Checkout Success" page displaying order details.
- Generate random coupon codes for users.

## Installation
1. Clone the repository: `git clone <repository_URL>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the Flask application: `flask run`

## Usage
1. Navigate to the project directory.
2. Run the Flask application.
3. Access the application through the provided URL.
